# hw01

\# 八皇后问题工程化实践（HW01）



\## 项目简介

基于 AI 协作完成八皇后问题的工程化实现，包含源码构建、自动化测试与 Bug 自主修复环节。



\## 实现思路

采用\*\*回溯法\*\*求解八皇后问题：

1\.  \*\*合法性校验\*\*：`is\_valid` 函数检查同列与对角线冲突，确保皇后放置位置合法；

2\.  \*\*递归回溯\*\*：`backtrack` 函数逐行尝试合法列，递归探索下一行，回溯时重置状态；

3\.  \*\*解收集\*\*：当遍历完所有行时，记录当前棋盘状态为一个完整解。
